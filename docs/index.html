AdminBro

Admin Bro is an Admin Framework for Node - your best Bro in app developement.

Currently it supports only MongoDB (via mongoose).

How it is built

It is totally separated from any particular nodejs framework. Because of that it can be easily integrated to almost every app.

Integration

Example integration for Hapi.js framework can be found here. This code uses Hapi.js integration plugin

Data model

graph TD A[<a href=/abstractdatabase.html>AbstractDatabase] -->|has many| B(AbstractModel) B --> |has many|C(AbstractInstance) B --> |has many|D(AbstractPorperty)

Most important part of the system is its data model. AdminBro can be integrated with multiple ORMs, that is why it has internal abstraction for handling multiple data models.

First of all base class is used to convert all supported database connections (mongodb, sql-like, etc) to list of databases which interits from AbstractDatabase. It utilises DatabaseFactory to construct correct database type.

Than each class which inherits AbstractDatabase can fetch all models present in the database. Each model inherits from AbstractModel

Each model has multiple properties with different types like String, Data, Number etc.

Finally particular database record/document is mapped to AbstractInstance
