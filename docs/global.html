ResourceFactory(resource, decoratoropt) → {BaseResource}

It changes raw resource object to a particular Resource implementation supported by AdminBro.

When Decorator class is given it allso attached it to the Resource. Otherwise it attaches BaseDecorator

Name Type Attributes Description
